# Aiden 的 AI 之旅 🗺️

一个 8 岁小朋友和爸爸妈妈一起学 AI 的亲子课程，共 12 课。
每一课 = 一个概念 + 一次动手 + 一件可保存的作品。

**👉 [打开冒险地图，开始学习](https://hiveronica2022.github.io/aiden-ai-journey/)**

## 内容

- `day-of-ai-course-roadmap.html` — 冒险地图：第 3–12 课总览 + 打卡进度
- `day-of-ai-*-lesson.html` — 每课的互动课件（第 1–12 课）
- `学习计划.md` — 给家长看的完整学习计划
