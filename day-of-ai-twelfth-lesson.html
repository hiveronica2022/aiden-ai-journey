任务 0

今天不上课，今天开发布会！🎉

从第一课到现在，你学了好多、做了好多。今天把家人请过来，办一场属于你的作品发布会，展示这一季所有的成果！

3 分钟

发布会前，先想一想

写一句你的想法，发布会就正式开始筹备啦。

家长开场白

“今天你是主角。这几个月你学的每样东西、做的每件作品，都要让大家看到。准备好当发布会主持人了吗？”

提前请家人：认真看、多提问、别敷衍。孩子第一次公开展示的反馈，会决定他想不想继续。

任务 1

作品大盘点

这是你这一季做过的所有作品。一件一件检查，准备好一件就打个勾。

8 分钟

我的作品清单（准备好就打勾）

把准备好的作品都勾上。没做完的也没关系，勾上做过的就行。

找不到某件作品？

  • 游戏和网页：打开 GitHub 作品博物馆就能找到。
  • 模型、画、故事：翻翻每课存的截图和记录。
  • 哪件没做，就当成“第二季再来做”，不用有压力。
“每一件作品，都是你动手做出来的，都值得骄傲。”
任务 2

编排演出顺序

发布会要有节目单！给每件作品选一个出场号码，排出你想要的顺序。

7 分钟

给作品排出场顺序

给你选好的作品各排一个号码，就有节目单啦。

排序小技巧

  • 把最得意的作品放最后，压轴登场。
  • 第一个选个简单好懂的，暖暖场。
  • 号码可以重排，想怎么调都行。
“好的发布会，像讲一个越来越精彩的故事。”
任务 3

练一段开场白

主持人要有开场白！填一填，下面会自动拼出你的开场白，对着它大声练两遍。

7 分钟
填一填上面三格，你的开场白会出现在这里。
三格都填好，开场白就完成啦。对着它大声练一练！

练习小提示

  • 站直，看着大家，慢慢说。
  • 先小声练一遍，再大声练一遍。
  • 说错了没关系，笑一笑接着说。
“大家好，我是 ___。请看我的作品！”
任务 4

发布会进行时 🎤

灯光准备好，主持人上台！按节目单一件件展示。每件作品用这个计时器，控制在 3 分钟左右。

现场

观众须知（请家人先读一读）

🎧 认真看每一件作品
🙋 至少问一个问题：“这是怎么做的？”“最难的是哪里？”
👏 每讲完一件，用力鼓掌
❤️ 别敷衍——你的认真，是给他最好的礼物
每件作品计时（3 分钟）
3:00
展示完至少一件作品后，点下面继续。

主持人小抄

  • 每件作品说三句：这是什么、怎么玩、我学到了什么。
  • 可以让家人上手玩一玩你的游戏。
  • 被问到不会的，就说“这是个好问题，我们一起看看”。
“慢慢来，你比你以为的更棒。”
任务 5

作品集主页剪彩 ✂️

你的 GitHub 作品博物馆，今天正式开馆！和爸爸妈妈一起，按下这个大大的剪彩按钮。

5 分钟

准备好了吗？

和家长一起打开你的 GitHub 作品集主页（那个有你卡通头像、一句话介绍和作品列表的页面）。全家一起数“三、二、一”，然后按下按钮！

🎊 作品博物馆正式开馆！🎊
和家人一起数三下，再按剪彩按钮。

这一刻的意义

  • 孩子的作品，从此有了一个真正属于他的“家”。
  • 把网址发给爷爷奶奶、好朋友——让更多人看到。
  • 钥匙先由你保管，13 岁生日再郑重交给他。
“这是你的博物馆。以后每做一件新作品，我们就来这里添一件收藏。”
