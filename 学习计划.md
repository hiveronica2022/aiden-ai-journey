# Aiden 的 AI 学习计划（8 岁版）

## 现状盘点

- ✅ 第一课：AI 侦探任务（认识身边的 AI，抓 AI 犯错）
- ✅ 第二课：什么是 AI（规则 vs 判断、算法、数据）
- 📄 第三课课件已备好：机器如何学习（训练数据、测试、偏差、Teachable Machine）
- 🎮 实践作品 1 号：俄罗斯方块游戏

## 总体理念

**每一课 = 一个概念 + 一次动手 + 一件可保存的作品。**

- 频率：每周 1 次，每次 45–60 分钟（8 岁孩子专注力上限，宁短勿长）
- 结构模板：5 分钟回顾上次 → 15 分钟新概念 → 25 分钟动手 → 10 分钟"存档仪式"（见 GitHub 部分）
- 家长三原则：
  1. **键盘在孩子手上**——哪怕打字慢，操作让他做
  2. **提示词让孩子口述**——用 AI 做东西时，让他说需求、你打字或他打字，学会"把想法讲清楚"本身就是最重要的 AI 技能
  3. **失败也存档**——做砸的作品同样记录，写上"哪里不行、为什么"

## GitHub 方案：四步走

> ⚠️ **重要前提**：GitHub 服务条款要求用户年满 13 岁。所以现阶段的正确做法是**家长注册并管理账号**，孩子在陪同下使用；等他 13 岁时把账号正式"移交"给他。可以把这一点变成仪式感："这是爸爸/妈妈帮你保管的博物馆，13 岁生日交钥匙。"

### 第 0 步（家长课前完成，约 20 分钟）

- 用家长邮箱注册账号，用户名建议中性一点（如 `aiden-builds`、`aiden-ai-journey`），将来移交不尴尬
- 开启两步验证
- 本文件夹已经是 git 仓库，直接作为第一个仓库推上去即可（仓库名建议用英文，如 `aiden-ai-journey`）

### 第 1 步（随第 6 课）：认识"作品博物馆"

- 给孩子讲概念用比喻：GitHub = 全世界程序员共用的**作品博物馆 + 时光机**，每次"存档"（commit）都会被永远记住
- 一起把俄罗斯方块放进仓库，README 由孩子**口述**、家长打字：我叫什么、这是什么游戏、怎么玩
- 让他自己点下发布按钮

### 第 2 步（随第 7–8 课）：GitHub Pages 上线 ★ 最大激励点

- 在仓库设置里开通 GitHub Pages，游戏和个人网页变成**真实网址**
- 把链接发给爷爷奶奶、同学家长——"别人真的能玩到我做的游戏"是这个年龄最强的驱动力
- 第 8 课的个人主页做成作品集入口，链到每个游戏

### 第 3 步（每课固定）：存档仪式（10 分钟）

每课最后 10 分钟固定做：

1. 作品文件放进对应文件夹
2. 截一张图或录 30 秒屏
3. 孩子口述三句话写进记录：**我做了什么 / 最难的是什么 / 下次想做什么**
4. commit 信息由孩子自己想（"给方块换了新衣服"这种也很好）

### 第 4 步（第 12 课前后）：作品集主页

- 建同名 profile 仓库（`aiden-builds/aiden-builds`），README 就是他的门面：头像（卡通的）、一句话介绍、作品列表
- 发布会那天正式"剪彩"

### 建议的仓库结构

```
aiden-ai-journey/
├── README.md          ← 作品总目录（孩子口述维护）
├── lessons/           ← 三个课件 html 移进来
├── games/
│   ├── tetris/        ← 俄罗斯方块 + README + 截图
│   └── my-game/       ← 第 7 课原创游戏

```

## 给家长的补充提醒

1. **课前一天试跑工具。** Teachable Machine、Quick Draw、AutoDraw 均为 Google 产品，国内网络直连不可用；GitHub 可访问但不稳定。每课前一天自己完整走一遍当天的工具。备选：Machine Learning for Kids、Code.org 的 AI 单元、腾讯扣叮。
2. **AI 聊天工具同样有年龄限制。** Claude、ChatGPT 条款不允许 8 岁儿童独立使用——用家长账号、全程陪同，顺带做内容把关。
3. **AI 家规（提前立好并讲给孩子）：** "AI 帮我们做我们还不会做的事（写代码），不代替我们正在练的事（作业、作文、算术）。" 第 10 课讲 AI 犯错时呼应此条。
4. **课表是菜单不是军令状。** 孩子对哪课上头就多停留，无感就跳过；唯一不变的是每课最后 10 分钟的存档仪式。
5. **立即做第一次本地 commit。** 本文件夹目前零提交，课件只存在这一台电脑上；不必等 GitHub 注册，先本地存档。
6. **英文界面当免费英语课。** 让孩子先猜按钮含义（"Train 是什么？"），不要代劳翻译。
7. **发布会提前"布置观众"。** 提前请家人认真玩、多提问，第一次展示的反馈质量决定孩子的持续动力。
